# # Dr.CRYPTO

블록체인을 활용한 의료 기록 관리 애플리케이션

데모영상 : <https://youtu.be/Oz8dfvZYVcw>


## # 개요

이더리움 블록체인 네트워크(Rinkeby)를 이용해 언제 어디서든 개인 진단서를 안전하게 발급받고, 타 의료기관으로 전송할 수 있는 애플리케이션


## # 주요기능

* 개인 의료 지갑 생성

* 병원 진단서 발급

* 타 의료기관으로 전송
